**Superstore Sales Dashboard (Tableau)**

**Project Overview**

This folder contains the Tableau workbook for an **Interactive Superstore Sales Dashboard** built to visualize key metrics and trends from a retail superstore dataset. The dashboard allows users to analyze:

- **Total Sales over time**
- **Sales by Category and Sub-Category**
- **categories performance over time** 
- **Top Categories** 

The dashboard is designed to be interactive, providing drill-down capabilities through filters such as **Category**, **Sub-Category**, and **Region**.

**Features**

- **KPI Cards** for:
  - Total Sales
  - Total Orders
  - Top Category
- **Line Chart** showing **Sales Over Time** (Month-Year format)
- **Bar Chart** showing **Sales by Category**
- **Donut Chart** showing **Sales by Sub-Category**
- categories performance over time 

**Data Source** : 

The dashboard uses the **Superstore dataset** from Tableau's sample data, which includes information on sales transactions, product categories etc.. 

   - The dashboard is already interactive. Click on the **Category** or **Sub-Category** filters to explore different segments of the data.

**Instructions for Use**

1. **Interacting with the Dashboard**:
   - Use the filters on the right to select different **Category** or **Sub-Category** values.
   - Hover over the charts for tooltips that provide additional details.
   
2. **Insights**:
   - The dashboard includes key insights, such as:
     - "Sales peaked in December 2021 across all categories."
     - "Office Supplies showed the highest profit ratio despite low sales."
     - "Technology continues to lead in sales volume and revenue."
   
**Key Visualizations**

1. **Sales Over Time**: A **Line Chart** showing sales trends month-by-month.
2. **Sales by Category**: A **Bar Chart** visualizing sales distribution across product categories.
3. **Sales by Sub-Category**: A **Donut Chart** showing sales breakdown by sub-categories.
4. **categories performance over time** :A stacked (area)chart for analyzing categories performance.

**Technologies Used**

- **Tableau Desktop**: Data visualization tool used to create the dashboard.
- **Superstore Dataset**: Pre-built retail dataset provided by Tableau for educational purposes.
