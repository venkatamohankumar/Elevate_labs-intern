This repository contains a Power BI project focused on financial analytics, using a finance dataset to provide insights into sales and profit performance.

 üìä Project Overview

The project includes the following key components:

 1. **Home Page**
- Created using Power BI features like buttons and page navigation.
- Showcases KPIs using cards.
- Provides navigation to three report pages:
  - Complete Report Page
  - Sales Overview
  - Profit Overview

 2. **Report Page**
- Displays tabular representation of the entire dataset.
- Visual enhancements:
  - Data bars for Sales and Profit%.
  - Color scaling for Profit Percentage.

3. **Profit Overview**
- Summary of total profit using DAX calculations.
- Key insights:
  - Highest profit recorded in **December 2014**.
  - Products with **low discounts** have the **highest profit**.
  - Includes tooltip showing sales for each profit range.

 4. **Sales Overview**
- Summary of total sales using DAX calculations.
- Key insights:
  - Highest sales recorded in **October 2014**.
  - Products with **medium discounts** have the **highest sales**.
  - Includes tooltip showing profit for each sales value.
