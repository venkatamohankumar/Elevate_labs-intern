# Shark Tank India Dashboard Analysis

Welcome to the **Shark Tank India Dashboard** developed by **Elevate Labs**. This project provides an in-depth, data-driven analysis of startup pitches, investor behavior, industry trends, and founder success patterns featured on the show *Shark Tank India*.

## Objective

The primary goal of this dashboard is to:
- Visualize key metrics such as average deal value, equity offered, and valuation.
- Identify industry-wise investment trends.
- Highlight investor preferences and participation frequency.
- Analyze founder success patterns (gender, team composition, geography).

## Tools & Technologies Used

- **Tableau** – for interactive dashboard visualization  
- **Excel** – for data cleaning and preprocessing  
- **jupyter**- Analyze founder profiles, funding stage success

## Key Insights

- **Top Industries:** Food, Education, and Beauty/Fashion attracted the highest investments.
- **Investor Leader:** Aman Gupta led with the highest number of deals and investment value.
- **Deal Types:** Equity-based deals were most common, with 39 deals including royalty models.
- **Founders:** Mixed-gender and couple-led startups showed higher success rates.
- **Hotspots:** Maharashtra, Delhi, and Karnataka were the leading regions by deal volume.
